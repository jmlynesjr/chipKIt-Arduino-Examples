ChipKIT UNO - Arduino/mpide example files

-----------------------------------------------------------------------------------------

RotaryEncoderISR4.pde - Sparkfun Rotary Encoder Driver - Interrupt Version

RotaryEncoderPoll.pde - Sparkfun Rotary Encoder Driver - Polled Version

SerialLCDTest.pde     - Sparkfun Serial LCD Display    - Serial Version (2 line x 16 char)

SerialLedTestI2C.pde  - Sparkfun Serial LED Display    - I2C Version (4 char)

SerialLedTestSER.pde  - Sparkfun Serial LED Display    - Serial Version (4 char)

SerialLedTestSPI.pde  - Sparkfun Serial LED Display    - SPI Version (4 char)

LCDEncoderEntry.pde   - Data entry via rotary encoder and 2x16 LCD display
